# 🧠 StepWise AI  
### Reasoning Diagnostic Engine powered by Local LLMs

StepWise AI is an intelligent system that analyzes *how* a user thinks — not just *what* answer they give.

Instead of grading answers, it diagnoses reasoning step-by-step, identifies mistakes, and provides targeted feedback to improve understanding.

---

## 🚀 Features

### 🧩 Learning Mode
- Detects **multiple reasoning errors**
- Identifies the **first wrong step**
- Explains **why it is wrong**
- Provides **hints instead of direct answers**
- Shows a **correct step-by-step solution**
- Gives a **confidence score**

---

### 📈 Markets Mode
- Analyzes macroeconomic events (e.g., Fed rate hikes)
- Builds **causal reasoning chains**
- Predicts impact on:
  - Equities
  - Bonds
  - Currency
  - Commodities
- Lists underlying **assumptions**

---

### ⚡ Local AI (No API Required)
- Runs entirely on **local models via llama.cpp**
- Supports models like:
  - Qwen
  - NVIDIA Nemotron (recommended)
- No API cost, no rate limits

---

## 🧠 Why This Matters

Most AI tools give answers.

**StepWise gives reasoning feedback.**

This is powerful for:
- Students learning math/science
- Interview prep (problem-solving)
- Debugging logical thinking
- Understanding *why* something is wrong

---
